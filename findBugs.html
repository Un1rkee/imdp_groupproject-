FindBugs guide

Preparation 

 

 

Download eclipse and open the eclipse

FindBugs Installation

1.Click "help - install new software", as shown below:

2.Click "add" and input "findbugs" and "location" in the pop-up box "name"“ http://findbugs.cs.umd.edu/eclipse ”Click "OK", as shown below:

3.Select the corresponding plug-in and click "next - > next - > finish"

4.After the installation, restart eclipse, right-click the project file or directory, and you will find that there are more findbugs menus, as shown in the following figure:

Of course, you can also directly fromhttp://download.csdn.net/detail/hailshao/6593725 Download, copy the file to your local eclipse plugins directory, and restart eclipse.
