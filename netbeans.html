Netbeans Guide

Preparation

 

 

1.JDK installed

Netbeans Installation

1.Download NetBeans on the Oracle website

2.Select the corresponding version download, follow the direction indicated by the arrow, that is, click the acceot license agreement, and then select the download address on the right side of windows x64 suitable for your computer

3.It is recommended to put the installation package on disk D, create new NetBeans - > NetBeans 8.1, double-click to install NetBeans

4.Remember to change the installation path and put it on disk d instead of selecting the system disk. After the installation, double-click the NetBeans icon on the desktop or open it during operation to enter the program and create a new project. You can download the corresponding plug-ins according to your own needs.

Plug-ins

When developing a Java Web project, you will find that you need to configure the server. At this time, you need to download the server plug-in and click Add Select the appropriate server to download, after installation, you can select the local domain, and then click Finish to develop the web project.

You can also choose your own needs to download the IDE with different plug-ins. Click https://netbeans.org , select different versions according to your own needs
